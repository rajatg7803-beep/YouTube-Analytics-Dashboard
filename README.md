**YouTube Analytics Dashboard**

This Power BI report gives an easy-to-understand view of YouTube channel performance. It highlights important metrics like total views, watch time, engagement levels, and best-performing videos. The dashboard helps identify what content works well and how the audience interacts with it.

**What the Dashboard Shows**

Overall channel stats (views, subscribers, watch time)

Performance of each video

Breakdown of traffic sources

Key engagement patterns

**Why It Was Created**

This dashboard is a simple project built to demonstrate Power BI skills such as data preparation, DAX, and interactive visualization.
